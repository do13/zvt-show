# zvt-show

ZVT ECR-Interface Protocol Analyzer — parses and decodes hex-encoded ZVT payment terminal messages (ECR-Interface ZVT-Protocol, up to revision 13.13).
Tool can be build as webserver or **Electron-based** ZVT protocol analyzer

Specification: https://www.terminalhersteller.de/downloads.aspx

## Build

```bash
npm run dev        # Run in development mode (hot-reload on file changes)
npm start          # Run as Electron desktop app (production mode)
npm run build      # Build Windows portable executable (electron-builder)
npm run lint       # Lint zvt.mjs with ESLint
```

For browser-based development (no Electron), run `start_webserver.bat` which starts `live-server`.
To prevent issues (the code must run on the web server; otherwise, cross-origin errors will occur).

There are no tests configured.
